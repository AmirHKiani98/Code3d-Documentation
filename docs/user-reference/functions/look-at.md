# lookAt

## Description

Makes the camera look at a specific point in the scene.

## Overloads

### Overload 1

Makes the camera look at a specific point in the scene.

```ts
lookAt(point: object | string, direction: object | string): void
```

#### Parameters

| Name | Type | Description |
| --- | --- | --- |
| `point` | `object` \| `string` | The point to look at, specified as a vertex object or its variable name. |
| `direction` | `object` \| `string` | The direction to look at, specified as a vector object or its variable name. |

#### Returns

`void`

#### Notes

None.
