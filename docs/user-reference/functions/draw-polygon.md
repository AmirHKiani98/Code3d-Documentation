# drawPolygon

## Description

Draws a polygon defined by a list of vertices in 3D space.

## Overloads

### Overload 1

Draws a polygon defined by a list of vertices in 3D space.

```ts
drawPolygon(vertices: array): sketch
```

#### Parameters

| Name | Type | Description |
| --- | --- | --- |
| `vertices` | [`array`](../objects/array) | An array of vertex objects that define the corners of the polygon. The last vertex will be connected to the first to close the polygon. |

#### Returns

[`sketch`](../objects/sketch)

#### Notes

None.
