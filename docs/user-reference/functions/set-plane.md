# setPlane

## Description

Defines a plane in 3D space using a point and a normal vector.

## Overloads

### Overload 1

Defines a plane in 3D space using a point and a normal vector.

```ts
setPlane(equation: string): void
```

#### Parameters

| Name | Type | Description |
| --- | --- | --- |
| `equation` | `string` | The equation of the plane in the form Ax + By + Cz + D. This means a plane with normal vector (A, B, C) that is D units from the origin. Basically, it is defined as Ax + By + Cz + D = 0. |

#### Returns

`void`

#### Notes

None.
