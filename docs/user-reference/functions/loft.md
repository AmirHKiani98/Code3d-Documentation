# loft

## Description

Creates a lofted 3D shape by connecting multiple cross-sectional profiles.

## Overloads

### Overload 1

Creates a lofted 3D shape by connecting multiple cross-sectional profiles.

```ts
loft(areas: array): body
```

#### Parameters

| Name | Type | Description |
| --- | --- | --- |
| `areas` | [`array`](../objects/array) | An array of 2D areas (closed sketches) to be lofted together, specified as sketch objects or their variable names. Order matters! |

#### Returns

[`body`](../objects/body)

#### Notes

None.
