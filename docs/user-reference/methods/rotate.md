# rotate

## Description

Rotates a body or sketch around an axis.

## Overloads

### Overload 1

Rotates a body or sketch around an axis.

```ts
rotate(item: string, axis: axis, angle: number): body | sketch
```

#### Parameters

| Name | Type | Description |
| --- | --- | --- |
| `item` | `string` | The body or sketch to rotate. |
| `axis` | [`axis`](../objects/axis) | The axis to rotate the body or sketch around. |
| `angle` | `number` | The angle to rotate the body or sketch around the axis. |

#### Returns

[`body`](../objects/body) | [`sketch`](../objects/sketch)

#### Notes

None.
