# Variable

## Description
A variable that will be used in the function defined by the user. It can be regarded as the arguments of a function, such as `func1(arg1, arg2)`

### Syntax
Variables should be defined in the following way at the beginning for the function:

```ts
define myVariable1: Type
define myVariable2: Type
...
```
where `myVariable` is the name of the variable being used and Type is the type of the variable.
